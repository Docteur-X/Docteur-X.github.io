Summary of Algebra Rules

An algebraic expression consists of numbers and letters. Letters in algebra act as placeholders for numbers. So algebraic expressions are numbers and letters that are numbers, so algebraic expressions are just numbers and numbers.

All Possible Calculations in Algebra

You can add two numbers together: \( 3 + 4 = 7 \)

You can multiply two numbers together: \( 3 \cdot 4 = 12 \)

You can subtract one number from another: \( 3 - 4 = 3 + (-4) = 3 + (-1) \cdot 4 = -1\)

You can divide a number by another (nonzero) number: \( \frac{3}{4} = 0.75 \)

Rules for Addition:

1. If you have a list of numbers added together, it doesn't matter in which order they add up.

Examples: \[ \begin{align} w+x+y+z = x + y + z + w = y + w + z + x = z + y + w + x \\ \frac{3a+b+c \cdot d}{3} = \frac{b + c \cdot d + 3a}{3} \\ ( xy + z^2 )^3 + \frac{z}{1 + \frac{1}{1+x}} + (3+a)(3+b)(3+c) = ( xy + z^2 )^3 + \frac{z}{\frac{1}{1+x} + 1} + (3+a)(3+b)(3+c) = ( xy + z^2 )^3 + \frac{z}{\frac{1}{x+1} + 1} + (a+3)(b+3)(c+3) = (a+3)(b+3)(c+3) + ( xy + z^2 )^3 + \frac{z}{\frac{1}{x+1} + 1} \end{align} \]

2. Adding 0 does 'nothing'.

Examples: \[ 3 + 0 = 3, 0 + \frac{3x+5}{2} = \frac{3x+5}{2} \]

3. A number subtracted by itself is equal to 0.

Rules for Multiplication

1. If you have a list of numbers multiplied together, it doesn't matter in which order they are multiplied.

1. Multiplying by 1 does nothing.

2. Multiplying by 0 destroys the number to 0

3. Multiplying by -1 creates the negative version of the number.

Rules for Expanding Brackets (Distributive Property)

x \cdot ( y + z ) = x \cdot y + x \cdot z
