Proof by Induction

Someone made a bold claim:

\[ \sum_{r=1}^n r^3 = \frac{1}{4}n^2(n+1)^2 \]

(In a clickable dropdown for the sum) \[ \sum_{r=1}^n r^3 \] means \[ 1^3 + 2^3 + 3^3 + \dots + n^3 \]. Add ref link to topic on summation.

Is the statement correct for n=1?

Let's say the statement is correct for n=1. Is it correct for n=2? If it is correct for n=2, is it correct for n=3?

If the statement is correct for n, is it correct for n+1?
