The Technique called
Completing the Square
to make \( \displaystyle ax^2 + bx + c = a \left( x + \frac{b}{2a} \right)^2 - \frac{b^2}{4a} + c \)

You can check that \( \displaystyle ax^2 + bx + c = a \left( x + \frac{b}{2a} \right)^2 - \frac{b^2}{4a} + c \).

(Dropdown) \[ \begin{align} a \left( x + \frac{b}{2a} \right)^2 &= a \left( x + \frac{b}{2a} \right)\left( x + \frac{b}{2a} \right) \\ &= a \left( \left(x + \frac{b}{2a}\right) x + \frac{b}{2a} \cdot \left(x + \frac{b}{2a}\right) ) \\ &= a \left( x^2 + \frac{bx}{2a} + \frac{bx}{2a} + \frac{b^2}{4a^2} \right) \\ &= a \left( x^2 + \frac{bx}{a} + \frac{b^2}{4a^2} \right) \\ &= a x^2 + bx + \frac{b^2}{4a} \end{align} \]

So \[ \begin{align} a \left( x + \frac{b}{2a} \right)^2 - \frac{b^2}{4a} + c &= a x^2 + bx + \frac{b^2}{4a} - \frac{b^2}{4a} + c \\ &= ax^2 + bx + c \end{align} \]

For example, if you want to turn \( \textcolor{red}{3}x^2 + \textcolor{orange}{1} x + (\textcolor{green}{-2}) \) into an expression that looks more like \( \textcolor{indigo}{p} ( x + \textcolor{brown}{q} )^2 + \textcolor{purple}{k} \), we use the above formula.

\[ \begin{align} \textcolor{red}{3}x^2 + \textcolor{orange}{1} \cdot x + (\textcolor{green}{-2}) &= \textcolor{red}{a}x^2 + \textcolor{orange}{b} x + \textcolor{green}{c} \\ &= \textcolor{red}{a}\left(x+\frac{\textcolor{orange}{b}}{2\textcolor{red}{a}}\right)^2 - \frac{\textcolor{orange}{b}^2}{4\textcolor{red}{a}} + \textcolor{green}{c} \\ &= \textcolor{red}{3}\left(x+\frac{\textcolor{orange}{1}}{2 \cdot \textcolor{red}{3}}\right)^2 - \frac{\textcolor{orange}{1}^2}{4 \cdot \textcolor{red}{3}} + (\textcolor{green}{-2}) \\ &= \textcolor{red}{3}\left(x+\frac{\textcolor{orange}{1}}{2 \cdot \textcolor{red}{3}}\right)^2 - \frac{25}{12} \\ \end {align} \]
